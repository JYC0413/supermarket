# 数学便利店收银员游戏 — 设计文档

**日期**：2026-05-13  
**目标年级**：二年级（刚学完九九乘法表）  
**教学目标**：巩固乘除法运算，培养"审题 → 圈关键词 → 作答"的解题习惯  
**使用场景**：课堂互动白板大屏，学生轮流上台，单人操作

---

## 1. 游戏概述

像素风便利店场景（参考星露谷物语风格）。玩家扮演收银员，顾客不断走进来咨询或结账，每位顾客提出一道数学应用题。学生通过触摸白板完成答题，赚取虚拟金币。

全班共享一个合作目标（班级金币总额），同时有一个匿名的单局最高记录榜，形成"台面上是合作、暗地里有竞争"的课堂氛围。

---

## 2. 回合结构

### 一个回合（一位同学上台）

1. 学生走到白板，回合计时开始
2. 顾客依次出现，每次屏幕上只有一位顾客
3. 每位顾客出一道题，走三步流程（见第3节）
4. 回合结束条件（两者先到为准）：
   - 计时到达上限
   - 当前局全部顾客已接待完 → 触发速度奖励

### 教师可配置项

| 参数 | 默认值 | 说明 |
|---|---|---|
| 回合时长 | 300 秒 | 可调整 |
| 每局顾客数 | 8 位 | 可调整 |

### 每位顾客的独立结局

- 答对：获得该顾客的全部收入，顾客满意离开
- 耐心归零：顾客生气离开，损失该顾客全部收入，**游戏继续**
- 顾客的耐心值完全独立，互不影响

---

## 3. 三步题目流程

每道题强制按顺序走完三步，①②之间无时间门槛，②③之间有圈词门槛。

```
顾客出现
   │
   ▼
① 读题  ──（立即可开始点击）──▶  ② 圈关键词
                                        │
                              圈够必要关键词后解锁
                                        │
                                        ▼
                                  ③ 输入答案
```

### 步骤①②：读题 & 圈关键词

- 题目以顾客对话形式出现在对话框
- 句子按意义短语预切块（如 `3个`、`6元`、`多少钱`），每块独立可点
- 连接词（"一共要付"、"买了"等）灰色显示，不参与点击
- 点击正确关键词：金色高亮，立即结算关键词奖金
- 点击非关键词：词块抖动后复位，**无任何惩罚**
- 圈够所有必要关键词后，数字键盘解锁

### 步骤③：输入答案

- 数字键盘出现（像素风格，与整体画面统一）
- 答对：结算基础奖励，顾客满意离开，进入下一位
- 答错：可重试，但耐心值按次递增扣除（见第4节）

---

## 4. 耐心值机制

每位顾客有独立耐心条，初始满值 100%。

### 自然消耗

| 阶段 | 消耗速率 | 说明 |
|---|---|---|
| 读题 & 圈词阶段 | −1% / 4秒 | 极慢，充分保证审题时间（满值约6分钟） |
| 数字键盘解锁后 | −1% / 2秒 | 稍快，暗示"该答了" |
| 键盘解锁 20 秒后仍未提交 | −3% / 2秒 | 加速警告，但不突兀 |

### 答错扣除（渐进式）

| 第几次答错 | 即时扣除 | 设计意图 |
|---|---|---|
| 第1次 | −15% | 还有机会，不慌 |
| 第2次 | −25% | 开始紧张 |
| 第3次 | −35% | 配合自然消耗，基本走人 |

速度快的学生勉强可以撑过3次尝试；节奏慢的两次错误后顾客大概率离开。

## 5. 计分与经济系统

### 每题收入构成

| 来源 | 金额 | 条件 |
|---|---|---|
| 正确答题基础奖励 | ¥30 | 答对即得 |
| 关键词奖金 | ¥8 × 正确词数 | 圈对一个词即结算，不等到答题 |
| 连胜加成 | 见下表 | 答对后触发检查 |

### 连胜奖励

- **连胜定义**：连续答对（中途任意一次答错即归零）
- 3连胜：+¥15
- 5连胜：+¥25（叠加在3连之上）
- 视觉反馈：火焰特效逐级增强（细节待定）

### 速度奖励

全部顾客接待完且计时未到：**+¥10 × 剩余每10秒**  
（剩60秒 = +¥60）

### 典型局面收入参考

| 局面 | 预期收入 | 说明 |
|---|---|---|
| 出色发挥 | ~¥490 | 全对，5连，提前30秒完成 |
| 正常发挥 | ~¥320 | 2题重试，3连一次，恰好用完时间 |
| 紧张失误 | ~¥180 | 3位顾客耐心耗尽，多次重试 |

### 计分展示

- **班级金币总额**：每回合收入累加，全班合作，持续显示在屏幕顶部
- **匿名单局最高记录**：本节课最高单局收入，随时可破，无需输入姓名

## 6. 题型与分布

### 每局固定配比（8题，随机排序）

| 题型 | 数量 | 难度标注 | 示例 |
|---|---|---|---|
| 纯乘法 | 3 | 无 | 我买3个面包，每个6元，一共多少钱？ |
| 有余数除法 | 3 | 无 | 我有20块钱，一根钢笔7元，能买几根？找多少钱？（分两步作答） |
| 乘加 | 1 | ⚡ 超难题 | 我买2个苹果和3个面包，苹果3元/个，面包6元/个，一共多少钱？ |
| 乘减/打折 | 1 | ⚡ 超难题 | 我买了4个本子，每个8元，满10元减3元，最后付多少钱？（打折只减一次；"买三送一"为另一种变体：每买3个送1个，实际等于少付一个的钱） |

**设计意图**：每位同学获得相同的题型构成，消除运气差异；复合题偶尔出现保持新鲜感，超难题标注管理预期并制造气氛。

### 关键词标准（各题型）

| 题型 | 必要关键词类型 |
|---|---|
| 纯乘法 | 数量词 × 1 + 单价 × 1 + 问句 |
| 有余数除法 | 总金额 + 单价 + 两个问句（几根 & 找多少钱） | 分两步作答：先答"几根"，再答"找多少钱" |
| 乘加 | 每种物品的数量 + 单价（各1对）+ 问句 |
| 乘减/打折 | 数量 + 单价 + 优惠条件 + 问句 |

关键词验证策略：**宽松匹配**——只要必要信息词被圈出即可通过，不要求圈出所有可能的词。

## 7. 界面布局

### 屏幕区域（从上到下）

```
┌─────────────────────────────────────────────────┐
│  🪙 班级: ¥3,240    ⏱ 2:18    👥 3/8    🏆 ¥610  │  ← HUD 顶栏
├─────────────────────────────────────────────────┤
│                                                 │
│          像素便利店场景（收银员 + 顾客）           │  ← 游戏场景
│                                                 │
├─────────────────────────────────────────────────┤
│  😊 顾客耐心 ████████████░░░░                   │  ← 耐心条
├──────────────────┬──────────────┬───────────────┤
│  ① 读题 ✓        │ ② 圈关键词 ✏ │  ③ 答题       │  ← 步骤指示
├─────────────────────────────────────────────────┤
│  顾客说：                                        │
│  我买了 [3个] 面包，每个 [6元]，一共要付多少钱？   │  ← 对话框
├──────────────────────────┬──────────────────────┤
│  答案输入框               │  7  8  9             │
│  [ 1_ ]                  │  4  5  6             │  ← 答题区
│  🔒 圈完关键词后解锁       │  1  2  3             │
│                          │  0  0  ⌫             │
└──────────────────────────┴──────────────────────┘
```

### 关键交互细节

- 关键词圈出时：金色高亮 + 浮动 `+¥8` 提示
- 误点非关键词：抖动动画，无视觉残留
- 步骤②完成时：步骤栏显示圈词奖金总额，键盘解锁动画
- 答对时：顾客表情变为满意，金币飞入顶栏班级总额
- 答错时：顾客表情变为不满，耐心条即时下降
- 连胜触发：火焰特效（3连小火苗，5连大火焰，具体待定）
- 超难题出现时：对话框旁出现 `⚡ 超难题！` 标签

## 9. 设计核心原则

1. **习惯优先**：三步流程不可跳过，但关键词奖金让"慢下来圈词"成为主动策略
2. **张力不焦虑**：耐心条慢慢消耗给足审题空间，答错才是真正的惩罚
3. **渐进失败**：答错不立即结束，渐进式惩罚保留翻盘空间
4. **集体荣誉感**：班级金币可见，每个人的贡献都算数
